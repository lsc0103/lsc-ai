三、功能优先级汇总

P0 - 必须有（MVP）
模块功能
Chat新建对话、历史对话、文本输入/输出、流式展示、文件上传、工作路径选择
Workbench动态开启、分屏展示、代码编辑器、数据表格、文件预览
User用户登录、基础权限
P1 - 应该有（完整版）
模块功能
Chat搜索对话、删除对话、重新生成、中断生成
Workbench多标签页、图表展示、AI 动态构建
Projects项目管理全部功能
Tasks定时任务、RPA 流程全部功能
Permissions角色管理、权限配置
Monitoring服务器监管功能
P2 - 可以有（增强版）
模块功能
Chat语音输入、对话归档
Workbench网页嵌入
UserSSO 登录
Settings主题设置、审计日志

依赖说明：
  • 用户认证 是所有功能的前置依赖，必须首先实现
  • 智能对话 是核心功能入口，触发 Workbench 展示
  • Workbench 承载所有任务执行结果的展示
  • Client Agent / RPA / Sentinel 是执行层，由 Workbench 触发
